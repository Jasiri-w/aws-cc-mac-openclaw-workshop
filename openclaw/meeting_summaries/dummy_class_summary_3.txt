# **Course Summary: Machine Learning Fundamentals (CS 3202)**

## **Course Overview**
CS 3202 introduces students to machine learning principles, algorithms, and applications. Students learn supervised and unsupervised learning techniques, feature engineering, model evaluation, and practical implementation using Python libraries like scikit-learn and TensorFlow.

## **Major Coursework & Deadlines**

### **1. Weekly Coding Assignments**
- **Description:** Python-based ML exercises covering regression, classification, and clustering.  
- **Weight:** 25%  
- **Deadlines:** Every Thursday at 11:59 PM  

---

### **2. Midterm Project**
- **Description:** Build and evaluate a predictive model on a provided dataset.  
- **Weight:** 20%  
- **Submission Date:** **October 24**  
- **Deliverables:**  
  - Jupyter notebook with analysis  
  - Model performance report  
  - Brief presentation  

---

### **3. Final Capstone Project**
- **Description:** Students select a real-world dataset and build a complete ML pipeline from data cleaning to deployment.  
- **Weight:** 35%  
- **Proposal Due:** **November 5**  
- **Final Presentation:** **December 16**  
- **Deliverables:**  
  - Complete code repository  
  - Documentation

### **4. Final Exam**

- **Weight:** 20%
